# analizador_de_gastos
Aplicación web desarrollada con Python y Flask que permite analizar gastos personales mediante inteligencia artificial. El usuario sube un archivo CSV con sus gastos y recibe un análisis detallado que incluye las categorías con mayor gasto, patrones de consumo, gastos no indispensables y recomendaciones personalizadas.
El análisis es generado por el modelo LLaMA 3.3 de Meta, accedido a través de la API de Groq. Los datos del CSV son procesados con Pandas y enviados como contexto al modelo para generar respuestas en lenguaje natural.
Tecnologías utilizadas: Python, Flask, Pandas, Groq API, LLaMA 3.3, HTML, CSS y JavaScript.

A web application developed with Python and Flask that analyzes personal expenses using artificial intelligence. Users upload a CSV file containing their expenses and receive a detailed analysis that includes the highest spending categories, consumption patterns, non-essential expenses, and personalized recommendations.
The analysis is generated by the Meta LLaMA 3.3 model, accessed through the Groq API. The CSV data is processed with Pandas and sent as context to the model to generate responses in natural language.
Technologies used: Python, Flask, Pandas, Groq API, LLaMA 3.3, HTML, CSS, and JavaScript.
